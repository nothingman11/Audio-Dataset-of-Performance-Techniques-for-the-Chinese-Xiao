# Audio-Dataset-of-Performance-Techniques-for-the-Chinese-Xiao
本数据集为系统性收录洞箫演奏技法的音频数据集（命名为“Chinese Xiao”）。录制使用96 kHz采样率、24-bit位深格式，采用六通道同步录音系统，使用6支共两种不同型号的麦克风对演奏过程进行拾音。数据集使用三支具有差异性的G调中国洞箫录制，音频数据共收录17种洞箫演奏技法（12种常规演奏技法，5种特殊演奏技法），共计19694个音频样本，文件总大小35.63GB，录音总时长17.06小时。 由于文件过大，为方便国内用户下载，在README文档底部附上百度网盘链接以及提取码。
