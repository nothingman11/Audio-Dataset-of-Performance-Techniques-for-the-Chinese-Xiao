# Audio-Dataset-of-Performance-Techniques-for-the-Chinese-Xiao

本数据集为系统性收录洞箫演奏技法的音频数据集（命名为“Chinese Xiao”）。在专业录音室录制。录制使用 96 kHz 采样率、24-bit 位深格式，采用六通道同步录音系统，使用 6 支共两种不同型号的麦克风对演奏过程进行拾音。数据集使用三支具有差异性的 G 调中国洞箫录制，音频数据共收录 17 种洞箫演奏技法（12 种常规演奏技法，5 种特殊演奏技法），共计 19,694 个音频样本，文件总大小 35.63 GB，录音总时长 17.06 小时。

### 📝 音频样本命名说明

所有音频文件采用如下命名格式：

`Xiao-Version-channel-technique-pitch-number.wav`

**字段解析：**
*   **Xiao**: 乐器类别标识，标记在该数据集所有音频命名之前。
*   **Version**: 洞箫编号标识（如 `A`, `B`, `C`），对应三支不同物理特性的 G 调洞箫。
*   **channel**: 麦克风通道标签，反映拾音位置（如 `Close` 表示近场麦克风）。
*   **technique**: 演奏技法英文译名（如 `Flutter tonguing`, `Harmonic`）。
*   **pitch**: 实际音高或音高范围（如 `G4`, `D5`, `A2-A3`）。若技法不涉及固定音高（如气冲音）则省略该标记。
*   **number**: 相同条件下录制的有效录音序号（从 1 开始）。

**示例：**
`Xiao-A-Close-Pitch bending-A3-2.wav`
> 表示：演奏者使用第一把洞箫（Xiao A）在 A3 音高上演奏压音技法，由近场麦克风（Close）录制的第 2 次有效录音样本。
