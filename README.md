# Audio-Dataset-of-Performance-Techniques-for-the-Chinese-Xiao
本数据集为系统性收录洞箫演奏技法的音频数据集（命名为“Chinese Xiao”）。录制使用96 kHz采样率、24-bit位深格式，采用六通道同步录音系统，使用6支共两种不同型号的麦克风对演奏过程进行拾音。数据集使用三支具有差异性的G调中国洞箫录制，音频数据共收录17种洞箫演奏技法（12种常规演奏技法，5种特殊演奏技法），共计19694个音频样本，文件总大小35.63GB，录音总时长17.06小时。 由于文件过大，为方便国内用户下载，在README文档底部附上百度网盘链接以及提取码。

#音频样本命名说明

所有音频文件采用如下命名格式：
Xiao-Version-channel-technique-pitch-number.wav
其中：Xiao为乐器类别标识，标记在该数据集所有音频命名之前；Version为该音频样本录制时所使用的洞箫编号标识（如 A、B、C，对应三支不同物理特性G调洞箫）；channel为麦克风通道标签，反映拾音位置（如Close表示近场麦克风）；technique为演奏技法英文译名（如Flutter tonguing、Harmonic）；pitch为录制时演奏的实际音高或音高范围区间（如G4、D5、A2-A3,），若技法不涉及固定音高（如气冲音）则省略该标记；number为相同条件下录制的有效录音的序号（从1开始），用于区分同一条件下的多次录制。
示例：Xiao-A-Close-Pitch bending-A3-2.wav
表示演奏者使用第一把洞箫（Xiao A）在A3音高上演奏压音技法，由近场麦克风（Close）录制的第2次有效录音样本。
